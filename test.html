What is Bench?

Bench gives you access to much of our internal benchmark data so that you can compare the products you're interested in without searching for an older review. Within a product category (e.g. CPUs) we tried to keep the testbeds as consistent as possible (e.g. same drives, memory sizes, etc...) so you can make reliable comparisons between products.

We currently allow CPU, SSD, Smartphone and GPU comparisons using Bench however we will be adding more categories in the future. You will notice that GPU Bench is split into two categories: GPU Bench 2011 and GPU Bench 2010. GPU Bench 2010 is a snapshot of the original GPU Bench, while the 2011 version includes newer GPUs, drivers and benchmarks. If you have any requests please email us directly. Any technical problems, please email our webmaster.
